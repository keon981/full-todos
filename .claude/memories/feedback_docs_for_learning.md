name: 學習過程寫入 docs
description: User prefers implementation plans written to docs/ directory as learning guides they can reference while coding
type: feedback
---

學習相關的實作計畫要寫進 docs/ 目錄，讓使用者邊做邊看邊記筆記。

**Why:** 使用者是透過做專案學習後端，需要一份可以隨時翻閱的參考文件，而不只是對話中的指引。

**How to apply:** 每個 LEARNING.md 任務開始時，先在 docs/ 建立對應的學習筆記文件（如 `docs/2-2-connect-postgresql.md`），包含步驟說明、概念解釋和筆記空間。
